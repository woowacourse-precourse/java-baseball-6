# 기능 목록

- 게임 시작
  1. 난수 생성
  2. 숫자 입력 받기 ( 조건 이외의 숫자일 경우 예외처리)
  3. 입력받은 숫자와 컴퓨터 난수와 숫자 비교
  4. 결과 출력
  5. 게임 종료키 입력
- 게임 종료
---

- 난수 생성 조건
: 1~9사이의 3개 랜덤 숫자 부여.
: 중복될 경우 제거 후 하나 추가로 생성함.

---

- 숫자 맞추는 조건 
: 자리와 숫자 모두 똑같은 경우 스트라이크.
: 같은 숫자가 있는 경우 볼
: 그렇지 않은 경우 낫싱

---

- 결과 출력 형식
: 스트라이크가 3개일 경우, `게임종료`
: 낫싱이 3개 일 경우, `낫싱`
: 볼, 스트라이크는 값이 0이상일 경우 출력.
