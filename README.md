# 미션 - 숫자 야구

## 🔍 구현 기능 List

- 게임 진행 구현
- 입력 구현
- 야구 기능 구현
- 결과 출력 기능 구현

### 🎯 게임 진행 구현

- 이중 반복문을 활용한 게임 진행
    - 전체 게임을 반복하기 위한 반복문
    - 게임 내부 로직을 반복하기 위한 반복문

### 🎯 입력 검증 구현

- 입력 기능을 두 가지 분리
    - 전체 게임 진행을 위한 입력 기능
    - 게임 내부 진행을 위한 입력 기능

### 🎯 야구 기능 구현

- Map을 활용해서 숫자의 위치 관리

### 🎯 결과 출력 기능 구현

- swtich문을 활용한 결과 출력
