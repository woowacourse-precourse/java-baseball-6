## 기능 목록

    1. 숫자 야구 게임을 진행할 램덤 숫자를 생성한다.

    2. 사용자에게 3자리 수를 입력 받는다.
        a. 입력의 유효성을 검증한다.

    3. 생성된 랜덤 숫자와 사용자의 입력을 비교한다.
        a. 스트라이크 개수를 계산한다.
        b. 볼 개수를 계산한다.

    4. 힌트를 출력한다. 

    5. 사용자가 정답을 맞힐 때까지 2~4의 과정을 반복한다.

    6. 사용자에게 게임 재시작 여부를 입력 받는다.
        a. 입력의 유효성을 검증한다.

    7. 재시작 여부에 따라 1~6 과정을 반복하거나, 프로그램을 종료한다.
