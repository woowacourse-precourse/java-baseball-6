1 ~ 9 사이의 서로 다른 숫자 3개를 뽑아 3자리 수를 정한다.

"숫자를 입력하세요 : "라는 문구를 출력한다.

숫자 입력 받기

3자리 수가 아닌 경우 -> 예외 출력

숫자가 아닌 경우 -> 예외 출력

중복된 숫자가 들어간 경우 -> 예외 출력

힌트 구하기

숫자가 같은 위치에 있다면? 스트라이크

같은 숫자, 다른 위치? 볼

1, 2번을 만족하는 숫자가 없으면? 낫싱

힌트 출력하기

정답 유무 판단하기

3 스트라이크인 경우 : 게임 종료

아니면? 다음 라운드로(2번으로 돌아가기)

게임이 종료가 되면, 재시작할지, 종료할지 물어본다.

입력받은 수가 1이면? 재시작

입력받은 수가 2이면? 게임 종료

그 외의 숫자 입력시 오류 발생
