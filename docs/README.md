# 구현할 기능 목록

## 1. 사용자 입력 받기

- 서로 다른 세 가지 숫자로 이루어 져야 함
- [예외] 입력 숫자가 정확히 3개여야 함
- [예외] 숫자 이외의 다른 문자가 들어 오지 않아야 함
- [예외] 같은 숫자가 들어 오지 않아야 함
- [예외] 숫자 0이 들어 오지 않아야 함
- 1을 입력하면 새로운 게임 시작
- 2를 입력하면 게임 종료

## 2. 숫자 야구 진행

- 임의의 서로 다른 세 가지 수 선택
- 사용자 입력과 비교

## 3. 결과 출력 하기

- 스트라이크, 볼 개수에 따른 결과 출력
- 스트라이크가 3개면 종료

## 4. 재시작 여부 확인

- 1일 경우 재시작, 2일 경우엔 종료
- [예외] 1, 2 이외의 값이 들어 오지 않아야 함
