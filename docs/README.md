### Computer

- [x] 서로다른 임의의 수 3자리 생성

```
[Valid] 중복된 숫자가 있는지 확인

[Valid] 1~9의 숫자인지 확인
```

### User

- [x] 1부터 9까지의 3자리 수를 입력받는다.

```
[Valid] 입력값이 숫자인지 확인

[Valid] 중복된 숫자가 있는지 확인

[Valid] 3자리 숫자인지 확인

[Valid] 1~9까지의 숫자인지 확인
```


- [x] 게임 종료 시 재시작(1), 종료(2) 중 하나의 숫자를 입력받는다.

```
[Valid] 입력값이 숫자인지 확인

[Valid] 1과 2의 숫자중 하나인지 확인
```

### Result

- [x] 컴퓨터의 값과 사용자의 입력값에 대한 결과 계산
  - [x] 힌트 출력
  ```
    같은자리 같은 숫자 : 스트라이크
    다른자리 같은 숫자 : 볼
    전혀 일치하지 않음 : 나싱
  ```
  -[x] 볼이 스트라이크 보다 먼저 출력되어야 한다.  

- [x] 재시작 또는 게임 종료
