## ⚾️ 숫자 야구 기능 목록

- 상대방(컴퓨터)의 랜덤 숫자 생성
    - 3자리 자연수 생성
    - 각 자리 숫자가 다른지 확인
- 사용자가 입력한 숫자 검증 후 저장
    - 사용자에게 입력값 요청
    - 서로 다른 3자리의 수가 아닌 입력값 예외처리
    - 사용자의 입력값 저장
- 결과 도출
    - 스트라이크, 볼 계산
    - 스트라이크, 볼, 낫싱 문구 출력
    - 정답인 경우, 종료 문구 출력
    - 오답인 경우, 사용자에게 입력값 요청
- 종료 조건 검사
    - 입력값이 1인 경우, 재시작
    - 입력값이 2인 경우, 종료
    - 입력값이 1 또는 2가 아닌 경우, 종료
