# 구현 기능 사항
- ## GameMaster
  ### 유저 입출력 관리
  - [ ] user input 처리
  - [ ] input validation
  - [ ] Game Interface와 통신 (input -> output)
  - [ ] output 출력
  - [ ] 게임 종료 혹은 재시작 관리

- ## NumberBaseball (Game Interface)
  ### 게임 진행 정보
  - [ ] 게임 초기화 로직
  - [ ] input에 따른 output 출력
    - [ ] input에 따른 결과 처리
    - [ ] 결과 값을 유저에게 줄 output으로 변환
    - [ ] 위 로직들을 추상화
  - [ ] 게임 진행에 필요한 (isOver, isProcessed) 등의 메서드
