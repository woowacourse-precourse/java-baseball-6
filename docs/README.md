# 구현 기능 목록

## 1. 초기값 설정 : init()

- 초기 컴퓨터의 정답값 랜덤으로 설정

## 2. 스트라이크, 볼 판정 : checkNum()

- checkStrike() : 스트라이크 수 판정
- checkBall() : 볼 수 판정
- checkAnswer() : 출력문 출력

## 3. 정답 판별, 종료 출력문 출력 : isEnd()

- 정답인 경우 종료 출력문 출력

## 4. 새로운 게임 시작 여부 판별 : checkNewGame()

- 1인 경우 초기화 후 새로운 게임 시작
- 2인 경우 게임 종료
