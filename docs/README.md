# 숫자야구게임 기능 명세서

## Player

- [X] 3자리 추측 값 입력받기
- [X] Note에 추측 값 담아서 상대에게 전달

## Computer

- [X] 생성자에 랜덤 숫자 생성하기.
- [X] Note가 볼/스트라이크/낫싱인지 확인
- [X] 볼 확인
- [X] 스트라이크 확인
- [X] 낫싱 확인

## Note

- [X] 3자리 String 중복 확인
- [X] 3자리 여부 확인
- [X] Note 객체 생성하는 메서드
- [X] toString()

## Application Main문

- [X] 게임 시작안내
- [X] 게임 진행
- [X] 게임 재시작여부
