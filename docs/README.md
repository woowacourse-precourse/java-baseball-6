## 구현 기능 목록
- 데이터 유효성 검사
  #### (사용자가 잘못된 값 입력시 "IllegalArgumentException" 발생시킨 후 종료)
    - 숫자만 입력했는지 확인
    - 세자리의 수를 입력했는지 확인
    - 서로 다른 수  3개 입력받았는지 확인
  

- 게임 시작, 진행, 종료 문구 출력


- 컴퓨터가 1 ~ 9까지 서로 다른 임의의 수 3개를 뽑아 리스트에 담는 함수


- 사용자가 입력한 수를 숫자 하나씩 형변환 하는 함수


- 사용자가 입력한 수가 정답인 수와 비교해서 결과(스트라이크, 볼, 낫싱)를 리턴하는 함수


- 게임을 재시작, 또는 종료시키는 함수
