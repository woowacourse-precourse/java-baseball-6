## 프로그램 진행 방식

1. 게임 시작 메세지를 출력한다.
2. 정답이 될 랜덤 숫자를 생성한다.
3. 사용자가 숫자를 입력한다.
4. 정답 숫자와 비교 후 결과를 출력한다.
   5-1. 만약 3스트라이크가 반환되면 게임을 종료한다.
   5-2. 아니라면 이전 3과 4 과정을 반복한다.
6. 사용자가 재시작(1) 혹은 종료(2) 숫자를 입력한다.
7. 재시작(1)시 위 2부터의 과정을 반복한다.
8. 종료(2)시 게임을 완전 종료한다.

## 주요 기능 목록

- 랜덤 수를 생성한다.
- 사용자 숫자를 입력받는다.
- 사용자 숫자 유효성을 검사한다.
- 두 수를 비교한다.
- 결과를 출력한다.

## 예외 목록

- 사용자 숫자 입력
    - 숫자의 자리수 확인(3자리)
    - 숫자의 범위 확인(1~9)
    - 숫자의 중복 확인
- 사용자 시작&종료 입력
    - 숫자 확인(1 혹은 2)
