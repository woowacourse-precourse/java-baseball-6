### 🎯1.1 2023/10/20 seunghune12
```
- 게임 시작 - application클래스를 실행하면 게임이 시작되고, '숫자 야구 게임을 시작합니다.'를 출력합니다. 
- 플레이어 숫자 입력 - '숫자를 입력해주세요 : '가 출력되면 플레이어는 숫자가 각각 다른 3자리의 숫자를 입력합니다.
   - 예외처리 - 만약 3자리의 숫자가 각각 다르지 않다면 IllegalArgumentException을 발생시킨 후 애플리케이션은 종료됩니다.
- 랜덤 3자리 숫자 생성 - 랜덤하게 세자리의 숫자를 생성합니다.
- 게임 진행 -
  - case 1 : 컴퓨터와 플레이어의 숫자 중 같은 자릿수에 같은 숫자가 있다면 일치하는 수만큼 숫자+스트라이크 출력
  - case 2 : 컴퓨터와 플레이어의 숫자 중 같은 자릿수가 아닌 다른 자릿수에 같은 숫자가 있다면 일치하는 수만큼 숫자+볼 출력
  - case 3 : 컴퓨터와 플레이어가 게임 중 스트라이크와 볼이 없다면 '낫띵' 출력
  - case 4 : 컴퓨터와 플레이어가 게임 중 같은 자릿수에 전부 같은 수가 있다면 '3스트라이크\n 3개의 숫자를 모두 맞히셨습니다! 게임 종료'를 출력
- 재시작 종료 분기 - 게임 진행 기능의 case4 가 실행되어 게임이 종료되면 '게임을 새로 시작하려면 1, 종료하려면 2를 입력하세요.'를 출력
  - case 1 : 재시작 - 1을 입력하고 enter를 누르면 첫 화면인 게임 시작부분으로 이동
  - case 2 : 게임종료 - 2를 입력하고 enter를 누르면 그대로 게임 종료!
  - case 3 : 예외처리 - 1과 2가 아닌 무언가가 입력되면 재시작 종료분기 재시작.
```



### 🎯v1.0 2023/10/20 seunghune12
```
1. 숫자 야구 게임을 시작합니다를 출력한다. 
2. 플레이어는 서로 다른 3자리의 수를 입력한다. 
3. 만약 서로 다른 3자리의 수가 아니라면 IllegalArgumentException을 발생시킨 후 애플리케이션은 종료되어야 한다. 
4. 컴퓨터가 111~999 사이의 숫자를 랜덤하게 저장한다.
5. 3개의 숫자를 서로 대조한다. 
6. 컴퓨터와 플레이어의 각각의 자릿수에 동일한 숫자가 아닌, 다른 자릿수에 동일한 숫자가 있는 경우 그 쌍만큼의 숫자와 볼을 출력한다.
7. 컴퓨터와 플레이어의 각각의 자릿수에 동일한 숫자가 하나도 없는 경우 낫띵을 출력한다.
8. 컴퓨터와 플레이어의 각각의 자릿수에 동일한 숫자가 있는경우 그 쌍만큼의 숫자와 스트라이크를 출력한다.
9. 컴퓨터와 플레이어의 각각의 모든 자릿수에 동일한 숫자가 있는경우 '3스트라이크'를 출력한 후 
'3개의 숫자를 모두 맞히셨습니다! 게임 종료 게임을 새로 시작하려면 1, 종료하려면 2를 입력하세요.를 출력한다.
10. 게임을 새로 시작하려면 1, 종료하려면 2를 입력하세요.를 출력한다.
11. 1을 입력하면 1로 돌아간다. 2를 입력하면 게임을 종료한다.
```
