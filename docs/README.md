# 우아한테크코스 -프리코스 1주차

## 기능 구현 목록

### 1. 유저

- 숫자 입력 받기
  1. 입력 값이 3자리 정수인지 확인
  2. 입력 값이 서로 다른 정수인지 확인
- 정답을 맞춘 뒤, 재시작/종료 입력 받기
  1. 입력 값이 1 또는 2인지 확인

### 2. 컴퓨터
- 중복되지 않은 랜덤한 값 정수 세자리 받기

### 3. 검증기

- 볼 검증하기
    1. 볼 갯수에서 스트라이크 갯수 빼기
- 스트라이크 검증하기
- 낫싱 검증하기
    1. 볼과 스트라이크 아무것도 없다면 낫싱
- 정답을 맞춘 뒤, 재시작 또는 종료 하기
    1. 유저에게 1과 2중 받아오기
