1. 야구 게임의 입력값을 받는 기능
    1. 정수임을 검증한다.
    2. 각 자리 수가 서로 다른 수임을 검증한다.
    3. 범위 안의 들어가 있음을 검증한다.

3. 야구 게임 기능
    1. 게임 초기 설정 하는 기능
    2. 랜덤 값을 생성하는 기능
    3. 게임을 진행하는 기능
    4. 볼이 몇개인지 확인하는 기능
    5. 낫싱인지 아닌지 확인하는 기능
    6. 게임 결과를 출력하는 기능
3. 게임을 재시작 할지 종료할지 입력받는 기능
    1. 프롬프트와 함께 값을 입력받는 기능
    2. 입력받은 값이 올바른지 검증하는 기능
