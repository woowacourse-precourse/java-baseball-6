
### 게임 단계
1. 게임 시작 안내를 한다.
2. 랜덤 숫자를 생성한다.
3. 사용자에게 숫자를 입력 받는다.
4. 입력 받은 숫자에 대해 스트라이크, 볼, 낫싱을 판단한다.
   1. 스트라이크일 경우, 게임 종료를 안내하고 게임 진행 상태에 대해 사용자 입력을 받는다.
   2. 아닐 경우, 3번으로 돌아간다.
5. 4-1에서 받은 값이 1일 경우, 2번으로 돌아간다. 2일 경우, 프로그램을 종료한다.


### 함수 정의

1. 베이스볼 숫자를 생성하는 함수
2. 입력받은 숫자에 대해 스트라이크 수를 판단하는 함수
3. 입력받은 숫자에 대해 볼 수를 판단하는 함수
