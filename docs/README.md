# 기능목록

- 컴퓨터는 숫자를 생성한다.
  - 1에서 9까지 서로 다른 임의의 수 3개를 생성
  - 3개의 수는 중복 X

- 사용자로부터 숫자를 입력받는다.
  - 서로 다른 3자리 수를 입력
  - 수의 범위는 1~9까지
  - 위 조건을 벗어날경우 *IllegalArgumentException*을 발생시키고 종료

- 컴퓨터와 입력 수를 비교한다.

- 볼, 스트라이크, 낫싱을 출력한다.
  - 볼 : 같은 수이나 다른 자리에 위치
  - 스트라이크 : 같은 수이며 같은 자리에 위치
  - 낫싱 :  입력된 수가 전부 다른 수

- 3스트라이크 일 경우 게임 종료 문구 및 재시작 여부를 묻는다.
  - 1을 입력시에 게임 재시작
  - 2를 입력시에 게임 종료
