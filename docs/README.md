1. 게임 시작
2. 컴퓨터에서 Random 숫자 생성
3. 숫자 입력 받기
   - 에러 체크
     - 3자리수 숫자가 정상적으로 입력되었나?
     - 입력 숫자 사이에 0이 들어있는가?
     - 숫자가 아닌 값을 입력받았는가?
     - 중복된 숫자 입력이 있는가?
4. 게임 로직 구현
   - 같은 자리에 같은 숫자가 있는 경우 : strike
   - 다른 자리에 같은 숫자가 있는 경우 : ball
5. 3 Strike가 되었을 경우 게임 종료
6. 게임 재시작 유무 판단
   - 새로 시작 : 1
   - 종료 : 2
   - 이외의 값 : 에러
