## 구현할 기능 목록

* 맞춰야 하는 서로 다른 임의의 수 3개를 정한다.

* 게임 시작 문구를 출력한다.

* 3자리 수를 입력받는다.
  * 입력 값이 중복일 경우, 예외 처리
  * 입력 값이 수가 아닐 경우, 예외 처리
  * 입력 값이 3자리가 아닐 경우, 예외 처리

* 입력 값에 따라 힌트를 출력한다.
  * 같은 수, 같은 자리인 경우 스트라이크
  * 같은 수, 다른 자리인 경우 볼
  * 모두 다른 수, 다른 자리인 경우 낫싱

* 3자리 수를 모두 맞출 때까지 입력과 힌트 출력을 반복한다.
  * 모두 맞출 경우, 게임을 종료한다.
  * 게임 종료 시, 재시작 혹은 종료를 선택하도록 한다.
  * 1 입력 시, 게임 재시작
  * 2 입력 시, 애플리케이션 종료
  * 이외의 값 입력 시, 예외 처리
