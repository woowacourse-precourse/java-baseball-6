

# 프리코스 1주차 숫자야구게임
## 프로그램 진행 순서
> 1. 랜덤한 숫자를 생성한다.
> 2. 사용자에게 숫자를 입력받는다.
> 3. 랜덤한 숫자와 사용자의 숫자를 비교한다.
> 4. 비교한 결과값을 출력한다.
> 5. 정답을 맞출때까지 2~4 반복하여 진행한다.
> 6. 정답을 맞추면 종료하고 다시 시작할지 물어본다.
> 7. 다시 시작하면 1~6 반복한다.
