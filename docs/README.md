# 프리코스 1주차 미션 - 숫자 야구

1. "숫자 야구 게임을 시작합니다." 출력

2. 1~9 사이의 중복 없는 3자릿수 난수 발생

3. "숫자를 입력해주세요: "출력

4. 사용자에게 숫자 입력받기

5. 입력값이 3자릿수가 아닌 경우 -> 예외출력

6. 입력값에 중복이 있는 경우 -> 예외출력

7. 입력값이 숫자가 아닌 경우 -> 예외출력

8. 난수와 입력값이 같은지 확인

9. 힌트 출력

10. 정답 유무 출력

11. 3스트라이크면 게임 종료

12. 게임 종료할지 재시작할지 입력받기

13. 입력받은 수가 1또는 2가 아니면 예외출력

14. 1입력 받으면 재시작 2면 게임 종료
