# 미션 - 숫자 야구
## 기능 목록 V2
### 야구 게임
- 게임을 실행한다.
- 게임을 진행한다.
  - 컴퓨터에게 서로 다른 3가지 숫자를 고르게 요구한다.
  - 유저에게 서로 다른 3가지 숫자를 고르게 요구한다.
    - 값이 서로 다른 3가지 숫자가 아닌 경우 IllegalArgumentException을 던진다.
  - 비교한다.
  - 비교 결과에 따라 다음 행동을 결정한다.
    - 3 스트라이크인 경우
      - 종료할지 재진행할지, 유저에게 요구한다.
        - 값이 1, 2가 아닌 경우 IllegalArgumentException을 던진다.
    - 3 스크라이크가 아닌 경우
      - 힌트를 생성한다.
      - 힌트를 출력한다.
- 게임을 종료한다.
- 게임을 재진행한다.
### 컴퓨터
- 서로 다른 3가지 숫자를 고른다.
### 유저
- 서로 다른 3가지 숫자를 고른다.
- 종료할지 재진행할지 결정한다.
