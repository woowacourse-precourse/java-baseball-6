# 구현할 기능 목록
### 1. StartCreate, StartCompare
랜덤함수를 작성하여 초기 셋팅을 진행합니다.
(1~9까지의 서로 다른 임의의 수 3개), (컴퓨터의 수와 사용자의 수 비교하기)


### 2. Ball
볼인 경우에 대한 기능 개발입니다.
(같은 수가 다른 자리에 있을 경우)

### 3. Strike
스트라이크인 경우에 대한 기능 개발입니다.
(같은 수가 같은 자리에 있을 경우)

### 4. Out
아웃인 경우에 대한 기능 개발입니다.
(같은 수가 1개도 없을 경우)

### 5. Clear
정답을 맞췄을 경우에 대한 기능 개발입니다.
