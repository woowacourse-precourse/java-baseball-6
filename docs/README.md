# baseball 프로젝트 기능 정리

1. 게임 시작 문구 출력
2. 컴퓨터 숫자 생성
    - 3자리 숫자 생성 (Randoms API)
    - 자연수 and 서로 다른 수 (조건)
3. 사용자 입력 
    - 입력 문구 출력
    - 3자리 숫자 입력받기 (Console API)
    - 자연수 (조건) -> 예외발생 -> 종료 
4. 계산 과정
    - 자리일치 && 값 일치 => 스트라이크
    - 자리불일치 && 값 일치 => 볼 
    - 볼, 스트라이크 개수 or 낫싱 출력
    - 정답 : 스트라이크 개수 \n 종료 문구
    - 오답 : 다시 입력받기
