# 숫자야구 게임 기능 명세서


## 랜덤 숫자 생성 기능
- 1에서 9까지 서로 다른 숫자 3개를 무작위로 생성
- 생성된 숫자는 컴퓨터에 저장

## 플레이어 입력 처리 기능
- 플레이어는 턴마다 3자리 숫자 입력
- 입력한 수 유효성 검사  
    - 유효하지 않으면, IllegalArgumentException 발생 후 애플리케이션 종료

## 숫자 비교 및 결과 확인
- 플레이어의 입력한 숫자와 컴퓨터가 선택한 숫자를 비교
    - 숫자와 위치가 일치하면 "스트라이크," 숫자만 일치하면 "볼," 모두 불일치하면 "낫싱"으로 결과를 판정
- 결과 출력

## 게임 종료 및 재시작
- 3개의 숫자를 맞힐 시 게임 종료
    - 게임 종료 시 옵션을 제공 (재시작 or 완전히 종료)
