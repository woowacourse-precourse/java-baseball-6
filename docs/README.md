# 기능 요구사항
### 기능 목록
- 입력 기능
  - 스트링 input 처리 기능(View)
  - 잘못된 입력 판별 기능(model)
- 게임 종료 기능(Controller)
- 게임 재시작 기능(Controller)
- 예외처리 기능(model)
  - 잘못된 입력
    - 길이가 3인 숫자가 아니다.
    - 입력값은 서로 다른 수여야 한다.
    - 게임종료/재시작 시 입력되는 수는 1또는 2여야한다.'
- 컴퓨터의 상대방 역할 기능 제공(model)
  - 1에서 9까지의 서로 다른 임의의 수 3개 제공
  - 스트라이크/볼/나띵 판별 기능
- 여러 출력기능(view)
