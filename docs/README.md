어떤 기능들이 있는가

1. 랜덤한 숫자를 생성한다
2. 사용자에게 숫자를 입력받는다
3. 랜덤한 숫자와 사용자의 숫자를 비교한다
4. 비교한 결과값을 출력한다
5. 정답을 맞출때까지 진행한다
6. 정답을 맞추면 종료하고 다시 시작할지 물어본다

어떤 객체들로 이루어져있는가?

- 사용자

1. 정답을 유추한다
2. 유추한 결과값을 보고 다시 정답을 유추한다
3. 정답을 맞추면 다시 할지 결정한다

- 컴퓨터

1. 랜덤한 숫자를 생성한다
2. 사용자의 유추값과 정답을 비교한다
3. 비교값을 알려준다
4. 정답을 맞출때까지 2~3 반복
