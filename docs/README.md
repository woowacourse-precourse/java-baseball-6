# 미션 - 숫자 야구

## ✔ 기능 체크리스트

- [ ] 게임 시작 문구 출력
- ### 컴퓨터
  - [ ] 숫자 초기화 
  - [ ] 3자리 순차적으로 1에서 9까지 서로 다른 임의의 수 3개 선택
- ### 사용자
  - [ ] 숫자 초기화
  - [ ] 입력된 숫자를 자릿수별로 배열로 변환
  - **예외처리**
    - [ ] 입력된 수가 3자리 수가 아닌 경우
    - [ ] 정수가 아닌 값 (예: 문자, 부동 소수점 수 등)을 입력
    - [ ] 입력된 수가 서로 다른 3개의 수로 이루어지지 않은 경우 (예: 222, 232)
- ### 입력된 숫자와 비교
  -  [ ] 스트라이크 개수 세기
  -  [ ] 볼 개수 세기
- ### 비교에 따른 결과 출력
  - [ ] 볼, 스트라이크 개수
  - [ ] 하나도 없는 경우
  - [ ] 3개의 숫자를 모두 맞힌 경우
    - [ ] 게임 종료 문구 출력


- [ ] 재시작 여부 문구 출력
