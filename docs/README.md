- 부연설명
  - 컴퓨터가 생성한 숫자와 사용자가 입력한 숫자를 비교해주는 기능
  - 경우의 수 : 스트라이크와 볼 수가 합쳐서 3을 넘어가거나, 1볼 2스트라이크 는 논리적으로 생길 수 없음을 명시.
    - 낫싱
    - 1스트라이크
    - 2스트라이크
    - 3스트라이크
      - 해당 경우, 게임을 종료해야하는 특이점 존재
    - 1볼
    - 2볼
    - 3볼
    - 1볼 1스트라이크
    - 2볼 1스트라이크
