# 기능목록

## 1. 컴퓨터(상대방)
    1. 랜덤하고 각 자리가 모두 다른 3자리 숫자 출력
    2. 게임플레이어 숫자와 자신의 숫자 비교
        1. 입력 숫자 유효성 검증
           (에러관리)사용자 오입력 시 IllegalArgumentException
    3. 게임 결과 출력

## 2. 게임 플레이어
    1. 각 자리가 모두 다른 3자리 숫자 입력
    2. 정답을 맞출 때까지 입력 반복

## 3. Application
    1. 컴퓨터, 게임 플레이어 생성
    2. 게임 실행
    3. 재시작 또는 완전 종료
