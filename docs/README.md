# 🔍 게임 흐름
1. 게임 시작 문구 출력
2. 컴퓨터 3자리 숫자 랜덤 생성
3. 사용자 3자리 숫자 입력
4. 컴퓨터 숫자와 사용자 숫자 비교하여 결과 계산
5. 틀렸을 경우, 힌트 출력 후 3번부터 반복 
6. 맞췄을 경우, 게임 종료문구 출력
7. 재시작 여부 확인
8. 재시작 할 경우 2번부터 반복
9. 종료할 경우 게임 완전 종료.

## 🚀 기능 요구 사항
1. [x] 컴퓨터 3자리 수 생성기능
2. [ ] 사용자 3자리 수 입력기능
3. [ ] 사용자 3자리 수 검증기능
4. [ ] 컴퓨터 숫자와 사용자 숫자 비교기능
5. [ ] 비교결과에 대한 힌트 출력 기능 
6. [ ] 재시작 여부 입력기능
7. [ ] 재시작 여부 검증기능
