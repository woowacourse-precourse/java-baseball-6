# 구현할 기능 

1. 서로 다른 임의의 수(1에서 9까지) 3개를 선택하는 기능
2. 게임 시작 전 문구를 출력하는 기능
3. 사용자에게 수를 입력받는 기능
4. 사용자가 잘못된 값을 입력할 경우 예외를 발생시키는 기능
5. 컴퓨터 수와 사용자가 입력한 수를 비교하는 기능
6. 사용자에게 힌트를 출력하는 기능
7. 게임을 종료 문구를 출력하는 기능
8. 게임 재시작 여부를 입력받는 기능
9. 애플리케이션을 중지하는 기능
