## 구현 기능 목록 🗒️

1️⃣ **정답 세팅**

- 1~9 사이의 서로 다른 숫자로 정답 구성

2️⃣ **사용자 입력**

- **(예외 처리)** 서로 다른 숫자로 입력을 구성하지 않은 경우

- **(예외 처리)** 정해진 자리수를 입력하지 않은 경우

3️⃣ **사용자 입력과 정답 비교**

- 입력 숫자를 바탕으로 정답 비교 (볼카운트 계산)

4️⃣ **정오답 여부에 따른 게임 지속**

- 정답(3스트라이크)을 맞혔을 경우 게임 종료

5️⃣ **재시작 여부 질의**

- **(예외 처리)** 1 또는 2 이외의 숫자를 입력했을 경우
