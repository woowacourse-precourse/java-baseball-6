## 기능목록

- [x] 게임 시작 문구를 출력한다.
- [x] 컴퓨터가 임의의 수를 생성한다.
- [x] 숫자를 입력 받는다.
- [x] 입력 받은 값이 올바른 값인지 확인한다.
    - [x] 3자리 숫자인지 확인한다.
    - [x] 숫자 0이 들어갔는지 확인한다.
    - [x] 3자리 모두 다른지 확인한다.
    - [x] 숫자인지 확인한다.
- [x] 입력 받은 수와 컴퓨터가 생성한 수를 비교한다.
- [x] 사용자에게 정보를 출력한다.
- [x] 결과가 같은 경우 재시작 여부를 입력 받는다.
