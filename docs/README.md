# 미션 - 숫자 야구

## ❓ 구현할 기능 목록
---

### 1. 무작위 서로다른 3자리의 수 생성

### 2. 사용자로부터 숫자 입력받기

### 3. 입력한 숫자가 3자릿수가 아닌경우, 0이 포함되어있는경우, 문자를 입력한 경우 오류처리

### 4. 생성된 수와 입력받은 수의 일치 여부
     - 생성된 숫자와 입력한 숫자가 일치 하는가
     - 생성된 숫자와 입력한 숫자의 자리가 일치하는가

### 5. 일치하는 숫자의 갯수 만큼 스트라이크 or 볼 or 낫싱 반환

### 6. 정답 여부 출력 (볼, 스트라이크, 낫싱)

### 7. 게임종료 후 진행여부
