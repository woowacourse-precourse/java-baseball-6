# 숫자 야구 미션

## 기능 구현 목록

### 컴퓨터 기능
- 1부터 9까지의 서로다른 3개의 숫자를 생성한다.
  - 3개의 숫자인지 검증한다.
  - 중복되는 숫자가 있는지 검증한다.

### 사용자 기능
- 3자리 숫자를 입력받는다.
  - 3개의 숫자인지 검증한다.
  - 중복되는 숫자가 있는지 검증한다.
- 게임 재시작 여부를 입력받는다.
  - 1 또는 2를 입력했는지 검증한다.

### 게임 기능
- 사용자 숫자와 컴퓨터 숫자를 비교해 "볼"개수를 계산한다.
- 사용자 숫자와 컴퓨터 숫자를 비교해 "스트라이크"개수를 계산한다.
- 사용자 숫자와 컴퓨터 숫자를 비교해 "낫싱"임을 판별한다.
- 3스트라이크가 아닌 경우 사용자에게 다시 숫자를 입력받는다.
- 3스트라이크인 경우 게임을 종료한다.
- 게임이 종료되면 게임 재시작 여부를 판별한다.
- 게임을 재시작 혹은 종료한다.
