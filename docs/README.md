## 숫자 야구

### 핵심 기능 : 컴퓨터 숫자와 사용자의 숫자를 비교해서 결과를 출력하기

#### 기능 목록

- 컴퓨터 숫자
    - [ ] 1~9사이의 서로 다른 3자리 숫자
        - [] 3자리인지 검증
        - [] 숫자인지 검증
        - [] 1~9 인지 검증
        - [] 1~9 사이 중 중복되는 숫자 있는지 검증

- 사용자 숫자
    - [] 1~9 사이의 서로 다른 3자리 숫자
        - [] 3자리인지 검증
        - [] 숫자인지 검증
        - [] 1~9 인지 검증
        - [] 1~9사이 중 중복되는 숫자 있는지 검증

- 비교하기
    - [] 컴퓨터 숫자와 사용자 숫자 비교하기
        - [] 같은 숫자가 있는지 비교하기
        - [] 같은 자리 인지 비교하기

- 결과 출력하기
    - [] 볼과 스트라이크 개수 세기
        - [] 같은 수가 다른 자리면 볼
        - [] 같은 수가 같은 자리면 스트라이크
        - [] 같은 수가 전혀 없으면 낫싱

- 재시작
    - [] 재시작 1, 종료 2
