# 미션 - 숫자 야구

## 🔍 기능 구현 목록

- [x] 서로 다른 임의의 수 3개 랜덤 선택
- [x] 사용자 입력 받기
  - [x] 예외 사항
    - [x] 입력값의 길이가 3이 아닌 경우
    - [x] 입력값의 첫글자가 0인 경우
    - [x] 입력값이 문자인 경우
- [x] 스트라이크, 볼 개수 세기
- [x] 스트라이크, 볼 개수에 따른 결과 문장 출력
  - [x] 숫자를 모두 맞출 때까지 반복
- [x] 숫자를 모두 맞추면 게임 종료
- [x] '게임 재시작 : 1, 게임 종료 : 2' 둘 중 하나를 입력받는다.
  - [x] 예외 사항
    - [x] 입력값이 1또는 2가 아닌 경우
    - [x] 입력값이 문자인 경우
- [x] 1을 입력 받을 경우 재시작
- [x] 0을 입력 받을 경우 게임 종료
