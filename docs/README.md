## 숫자 야구 - 구현할 기능 목록

1. **게임 시작**
    - 게임 시작 문구 출력

2. **컴퓨터 숫자 생성**
    - 1-9 사이의 숫자 중 랜덤한 숫자 3개 생성

3. **사용자 입력**
    - 사용자로부터 3자리 숫자 입력 받기
    - 입력 값 검증 (3자리 숫자인지, 중복된 숫자가 없는지 확인)
    - 사용자 입력을 숫자 리스트로 변환

4. **결과 계산**
    - 스트라이크와 볼의 개수 계산
    - 게임 결과 메시지 생성 (스트라이크, 볼, 낫싱)

5. **게임 결과 출력**
    - 계산된 게임 결과 메시지 출력

6. **게임 종료 확인**
    - 3스트라이크인 경우 게임 종료
    - 그렇지 않은 경우 다음 턴으로

7. **게임 재시작/종료**
    - 사용자에게 게임 재시작 또는 종료 여부를 묻는 입력 받기 (1 또는 2)
    - 입력에 따라 게임 재시작 또는 종료


## 구조

### 1. User 클래스
- 사용자로부터 3자리 숫자를 입력 받는 기능
- 사용자 입력을 숫자 리스트로 변환

### 2. Result 클래스
- 스트라이크와 볼의 개수를 계산
- 게임이 끝났는지 (3스트라이크인지) 확인
- 결과 메시지를 출력하는 기능 (스트라이크, 볼, 낫싱)

### 3. BaseballGame 클래스
- 컴퓨터의 숫자를 생성
- 사용자의 입력과 컴퓨터의 숫자를 비교하여 결과를 반환

### 4. Application  클래스
- 게임 시작 문구 출력
- 게임의 각 턴을 실행
- 게임 종료 여부를 확인
- 게임 결과를 출력
- 게임 재시작 또는 종료
