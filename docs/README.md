## 🚀 기능 요구 사항 목록

### 1. Computer

- [ ] 랜덤의 정답 숫자 생성
    - `camp.nextstep.edu.missionutils.Randoms의 pickNumberInRange()` 사용

### 2. Player

- 입력 : `camp.nextstep.edu.missionutils.Console의 readLine()` 사용

- [ ] 도전 숫자 입력
- [ ] 재시작 or 종료 숫자 입력

### 3. 힌트 제공

- 같은 수가 같은 자리 : 스트라이크
- 같은 수가 다른 자리 : 볼
- 같은 수가 전혀 없음 : 낫싱

### 4. 각종 constant, MESSAGE

- [ ] 게임 시작/힌트/종료 메세지
- [ ] 예외 처리 메세지
    - 숫자가 아닌 경우
    - 3자리가 아닌 경우
    - 중복된 숫자를 입력한 경우
