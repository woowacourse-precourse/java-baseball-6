## 기능 요구 사항
- 입력
  - 사용자가 서로 다른 세 자리의 수를 입력한다
  - 예외 처리
    - 0이 입력됨(1-9까지 숫자가 아님)
    - 세 자리 숫자가 아님
    - 세 자리 수가 동일함
- 출력
  - 게임을 시작한다 
  - 스트라이크, 볼, 낫싱을 표시한다
  - 재시작 또는 종료한다
- 실행 동작
  - 컴퓨터가 임의의 수 3개를 선택한다
  - 컴퓨터와 사용자의 수를 비교한다
    - 같은 수가 같은 자리: 스트라이크
    - 같은 수가 다른 자리: 볼
    - 같은 수가 존재하지 않음: 낫싱
