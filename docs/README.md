입력
세자리 숫자를 입력받아야 한다
-> 다른 문자를 받으면 예외 처리를 해야함. -> IllegalArgumentException

재시작 종료를 구분하는 1과 2의 수

작동
1.랜덤으로 세자리 숫자를 생성해야한다.
2. 배열에 숫자를 넣기
3. 비교하기
4. 스트라이크랑 볼 판정기능
5. 3스트라이크시 게임 혹은 재시작(숫자 재생성) 기능

출력
1. 숫자야구 시작
2. 숫자 입력받기
3. 결과 출력
4. 게임종료, 재시작 문구

예외 상황
1. 숫자안에 같은 숫자가 있을때
2. 세자리 숫자 초과거나 미만일 경우
3. 숫자말고 다른 것을 입력했을 경우
