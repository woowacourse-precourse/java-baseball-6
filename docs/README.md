## 숫자 야구 게임

### 구현 기능 목록

- 플레이어
    - 컴퓨터
      - 서로 다른 3개의 수 생성
    - 사용자
      - 서로 다른 3개의 수 입력
      - [예외] 4개 이상의 수를 입력한 경우 IllegalArgumentException을 발생
      

- 게임 진행
    - 컴퓨터가 임의의 수 3개 생성 
    - 플레이어가 임의의 수 3개 입력
    - 게임 결과 계산
    - 3개의 숫자를 모두 맞힐 때까지 반복
    - 게임 재시작 or 종료   
  
  
- 게임 재시작 및 종료 
  - 1: 재시작, 2: 종료
