## 2. 상수 데이터 정의

### 출력 문구

    - 시작 시 문구 : "숫자 야구 게임을 시작합니다."
    - 입력 관련 문구 : "숫자를 입력해주세요 : "
    - 힌트 문구 : "x볼 y스트라이크"
    - 정답 문구 : "3개의 숫자를 모두 맞히셨습니다! 게임 종료"
    - 재시작/종료 문구 : "게임을 새로 시작하려면 1, 종료하려면 2를 입력하세요.

### 상수 데이터

    - 숫자의 자리수 : 3
    - 랜덤 숫자의 각 자리의 수의 범위 : 1~9
    - 게임 재시작 flag : 1
    - 게임 종료 flag : 2
