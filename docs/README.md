# Number Baseball Game
| 우아한테크코스 프리코스 1주차 미션 구현

### 기능 구현 흐름
1. 게임의 큰 Flow 주석 및 세부 내용 작성
2. 주요한 기능을 하는 함수 뽑아내기
3. 주요 알고리즘 구현
4. 테스트
5. 메소드 추출 (코드를 깔끔하게 만들기)

### 구현할 기능 목록
- 점수 계산 알고리즘
    - 스트라이크, 볼 개수. 없으면 '낫싱'
- 라운드 결과 계산 알고리즘
  - 사용자가 결과를 맞췄는지 여부
- 랜덤 숫자 도출 알고리즘
  - camp.nextstep.edu.missionutils.Randoms의 pickNumberInRange()를 활용한 3개의 랜덤 숫자 도출
