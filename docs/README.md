# 구현할 기능 목록

- `Game`
    - 게임 흐름 제어
    - 게임 메시지 출력
- `Computer`
    - 컴퓨터가 생각한 랜덤 숫자 결정
- `Player`
    - 플레이어가 추론한 숫자 입력
    - 다음 게임 진행 여부 입력
- `Referee`
    - 컴퓨터가 생각한 숫자와 플레이어가 추론한 숫자에 대한 볼, 스트라이크, 낫싱 판정
- `Result`
    - `Referee`가 판정한 결과
- `Validator`
    - 플레이어 추론 값 검증
        - 입력 값의 크기 검증
        - 유효 숫자 범위 검증
        - 중복 숫자 검증
    - 다음 게임 진행 여부 값 검증
        - 유효 입력 값 (1 또는 2) 검증
