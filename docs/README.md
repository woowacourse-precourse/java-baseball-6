## Base Ball Game 기능목록

#### 게임 시작 기능

- [x] 시작 문구 출력

#### 정답 입력 기능

- [x] 숫자 입력 기능
- [ ] 입력이 숫자가 아닌 경우에 대한 예외 처리
- [x] 입력이 3자리가 아닌 경우에 대한 예외처리
- [ ] 입력이 서로 다른 숫자가 아닐경우에 대한 예외처리

#### 정답 판별 기능

- [x] 랜덤한 숫자 생성
- [x] 자리가 아닌 숫자만 맞힌 경우에 대해 볼 출력
- [x] 자리와 숫자가 맞는 경우에 대해 스트라이크 출력
- [x] 숫자를 모두 맞춘 경우에 결과 출력

#### 게임 재시작, 종료 기능

- [x] 숫자 입력 기능
- [x] 입력이 1일때 종료
- [x] 입력이 2일때 재시작
- [ ] 숫자가 아닌 경우에 대한 예외처리
- [ ] 1,2 외에 다른 숫자가 들어오는 경우에 대한 예외처리
- [ ] 입력이 없는 경우에 대한 예외처리
