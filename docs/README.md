# 구현 기능 목록



##  1. **Host**

- \*게임 참여자\*를 호출한다.
- 참여자들을 각자의 방식에 맞게 \*준비 상태\*로 만든다. 
- 게임을 시작하고 게임 결과를 출력한다.
- 게임 결과에 따라 게임을 \*조정\*한다.



## 2. **Computer**
 - 서로 다른 3자리의 수를 생성하기.
 

## 3. **Player**

- 서로 다른 3자리의 수 입력 받기.
- 입력이 잘못 됐다면 애플리케이션 종료 시키기.

<br>
<br>
*게임 참여자* : Computer, Player<br>
*준비 상태* : 서로 다른 3자리 수를 리스트에 담기<br>
*조정* : (1) 3스트라이크면 게임 종료 및 재시작/종료 구분 (2) 3스트라이크가 아니라면
Player를 준비상태로 만들기.
