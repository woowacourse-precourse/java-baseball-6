### 1. 설계

디자인 패턴 중 하나인 MVC 패턴을 채택하여 프로그램을 설계합니다.

- Controller
    - 유저에 대해서 입출력을 View와 Service에게 전달합니다.
    - 입력 값에 대해서 오류를 검증하고 예외를 던집니다.
- Service
    - 게임의 핵심로직에 대한 책임을 가집니다.
    - 볼과 스트라이트의 갯수를 확인하여 Controller에게 전달합니다.
- View
    - 사용자 화면에 게임 화면을 보여주는 책임을 가집니다.
