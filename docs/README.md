숫자야구
1~9 까지 서로 다른 수로 이뤄진 3자리 수 맞추기

기능 요구사항
1. 1~9까지 서로 다른 수로 이루어진 3자리 수 맞추기
2-1. 같은 수일 경우 스트라이크,
2-2. 다른 자리일 경우 볼,
2-3. 같은 수가 없다면 낫싱,
3. 컴퓨터가 1~9 랜덤 수 선택
4. 3개 수 모두 맞추면 종료
5. 잘못된 값 입력하면 오류 발생 후 종료
 5-1. 3자리 미만, 3자리 초과 입력
 5-2. 숫자 외 다른 문자 입력

프로그래밍 요구사항
1. jdk17 버전
2. build.gradle 파일 변경 x
3. 프로그램 종료시 'system.exit'호출 x
4. 테스트 모두 성공
5. 파일, 패키지 수정 및 이동 x
