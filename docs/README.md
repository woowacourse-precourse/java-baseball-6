1. 게임시작
2. 랜덤 세자리 수 생성(1~9 서로 다른 임의의 수) -> pickNumberInRange()
3. 사용자의 입력 -> readLine()
[예외] 잘못된 값 입력 -> IllegalArgumentException
- 3자리 자연수가 아닌 경우
- 1~9까지의 숫자 아닌경우
- 0 이하의 음수의 경우
- 숫자에 중복 값이 있는 경우
4. 숫자 비교
4-1. 자리와 숫자가 맞음 -> 스트라이크
     자리는 틀리고 숫자만 맞음 -> 볼
     자리도 틀리고 숫자도 틀림 -> 낫싱
5. 3스트라이크 시 게임 종료 문구 출력
6. 종료 조건 입력
6-1. 1 : 입력 재시작
6-2. 2 : 게임 종료
[예외] 1과2가 아닌 값 입력할 경우


구현방식
controller
- 전반적인 게임 진행
model
- 컴퓨터 랜덤 숫자 생성
- 사용자 입력 숫자 예외처리
util
- 컴퓨터와 입력 숫자값 비교
view
- inputview : 사용자의 입력을 필요로 하는 문구
- outputview: 사용자에게 결과값 출력 문구
