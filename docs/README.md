게임 기능
1. 같은 수가 있는지 확인하는 기능
2. 같은 자리에 있는지 확인하는 기능 --> 같은 수가 있는지 같이 확인됨
3. 볼의 개수를 세는 기능
4. 스트라이크의 개수를 세는 기능
5. int 타입을 List<Integer> 타입으로 변환하는 기능
6. 컴퓨터 수를 만들어주는 기능
7. 사용자 번호 받아오는 기능, 재시작 번호 받아오는 기능, 정답 맞췄는지 확인하는 기능 분리 


사용자 입력 예외 처리
1. 3자리 숫자가 맞는지
2. 중복된 값이 있는지
3. 게임 재시작 번호가 1, 2 이외의 값이 들어오진 않았는지

공백 제거 기능

사용자가 보는 화면 (view)
<출력 화면>
1. 게임 시작 문구
2. 게임 결과 문구

<입력 화면>
1. 숫자 입력 문구
2. 게임 재시작/종료 입력 문구
