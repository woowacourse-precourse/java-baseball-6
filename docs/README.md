    - [x] 공의 개수가 3개 초과인지 확인합니다.
    - [x] 공의 개수가 3개 미만인지 확인합니다.

    - [x] 입력된 숫자는 유효성 검증 후, input됩니다. (오류가 발생하면 IllegalArgumentException이 발생됩니다.)
