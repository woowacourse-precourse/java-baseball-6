# 숫자 야구 기본 기능
1. 서로 다른 3자리 수는 컴퓨터가 랜덤으로 생성한다.
2. 사용자는 서로 다른 3자리 숫자를 입력한다.
3. 같은 수 같은자리에 있으면 스트라이크, 다른 자리에 있으면 볼, 같은 수가 전혀 없으면 낫싱이 출력된다.
4. 3스트라이크일 경우 게임이 자동으로 종료된다.
5. 게임이 종료 되었을 때 1을 입력하면 새 게임, 2을 입력하면 완전히 종료된다.

### 제출 전 확인사항

- [X] JDK 17버전에서 실행 가능한가

- [ ] 프로그램 실행 시작점이 Application의 main()인가

- [ ] Java 코드 컨벤션 가이드를 준수했는가

- [ ] 종료시 System.exit()를 호출하지 않았는가

- [ ] 모든 테스트를 통과했는가

- [ ] Randoms 밒 Console은 camp.nextstep.edu.missionutils에서 제공한 API를 사용하였는가

- [ ] Random 값 추춘은 pickNumberInRange()를 사용하는가

- [ ] 사용자 입력값은 readLine()을 이용하는가
