게임시작
1. 랜덤 세자리 수 생성(1~9 서로 다른 임의의 수) -> pickNumberInRange()
2. 사용자의 입력 -> readLine()
2-1. 자리와 숫자가 맞음 -> 스트라이크
     자리는 틀리고 숫자만 맞음 -> 볼
     자리도 틀리고 숫자도 틀림 -> 낫싱
[예외] 잘못된 값 입력 -> IllegalArgumentException
- 3자리 자연수가 아닌 경우
- 1~9까지의 숫자 아닌경우
- 0 이하의 음수의 경우
- 숫자에 중복 값이 있는 경우
3. 3스트라이크 시 게임 종료 문구 출력
4. 종료 조건 입력
4-1. 1 : 입력 재시작
4-2. 2 : 게임 종료
[예외] 1과2가 아닌 값 입력할 경우
