1. 애플리케이션을 실행한 경우

게임 종료조건:
1. 사용자가 3개의 숫자를 맞힌 경우

애플리케이션 종료조건:
1. 게임실행중 또는 게임종료 후 사용자가 잘못된 값을 입력한 경우
2. 게임 종료 후 사용자가 2를 입력했을 경우
