## 기능 요구 사항
### 입력
- [x] 3자리 수 입력받기
- [x] 1 입력 시, 게임 재시작
- [x] 2 입력 시, 게임 종료
### 출력
- [x] 게임 시작 문구 출력하기
- [x] 입력한 수에 대한 결과 표시하기
  - [x] 하나도 맞추지 못한 경우, '낫싱' 출력
  - [x] 3개의 숫자를 모두 맞춘 경우, '3개의 숫자를 모두 맞히셨습니다!' 출력 후, 게임 종료
  - [x] 볼,스트라이크로 결과가 나타나는 경우
### 예외처리
- [x] 잘못된 값이 입력될 경우, `IllegalArgumentException` 발생 후 프로그램 종료
### Computer
- [x] 랜덤 수 생성하기
