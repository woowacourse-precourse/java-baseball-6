# 숫자 야구

## 개요
- **입력 숫자 세 자리**를 크기가 3인 배열에 저장해 정답과 비교한다.
- 비교 과정에서 상대(컴퓨터)는 유저에게 볼, 스트라이크, 낫싱으로 힌트를 준다.
- 정답을 맞췄다면 거기서 게임은 종료되며 재시작 여부를 선택할 수 있다.

## 순서
1. 상대방(컴퓨터)는 게임 시작과 동시에 **하나의 랜덤한 숫자**를 가짐.
2. 정답이 맞을 때까지, 유저(나)는 **질문(입력)을 반복**.
3. 정답을 맞았다면, 재시작 선택권이 주어짐.

## 유의 사항
- 잘못된 입력에 대해 `IllegalArgumentException`를 출력.
- 예를 들어, 
  - 질문을 할 때, 네 자리의 정수를 입력
  - 재시작 시에 '3'을 입력하는 경우.
- 재입력은 불가능하며 그 자리에서 프로그램이 종료됨.

## 기능 목록
- 정답 생성
- 입력 오류 검사
- 정답과의 비교
- 힌트 출력
- 재시작 여부

### `Game` 클래스: 숫자 야구 게임

- **멤버 변수**
  - `ball_Count`: 볼 개수
  - `strike_Count`: 스트라이크 개수
  - `rightAnswer[]`: 정답을 저장하는 배열

- **멤버 함수**
  - `generateRightAnswer`: 게임을 시작하면 랜덤한 세 자리 숫자를 만들어 정답 숫자를 만듭니다.
    - `checkDuplication`: 각 자리의 숫자가 중복되지 않도록 검사합니다.
  - `play`: 다음의 함수들을 통해 게임을 진행합니다.
    - `callArray`: 유저의 질문(입력값)을 배열에 입력합니다. 이 배열은 질문을 정답과 비교하기 위해 사용됩니다.
    - `compareAnswer`: 유저가 질문(입력값)을 하면 정답과 비교하여 볼과 스트라이크 개수를 계산합니다.
    - `showHint`: 유저의 질문(입력값)에 대한 힌트를 출력합니다. 힌트는 볼과 스트라이크 개수를 알려주고, 정답을 찾는 데 도움을 줍니다.
    - `gameSet`: 3스트라이크일 때 게임을 마칩니다.
    - `reStart`: 게임이 끝나면 유저는 재시작 여부를 선택할 수 있습니다.

### `Application` 클래스: 게임 실행
- `Game.play`를 호출하여 게임을 시작한다.
