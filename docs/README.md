# 숫자 야구 게임 구현할 기능 목록

## 시작
숫자 야구 게임을 시작합니다 문구 출력

## 숫자 생성
1부터 9까지 서로 다른 3개의 랜덤 숫자 생성

## 사용자로부터 숫자 입력받기
사용자로부터 3자리의 숫자 입력받기
서로 다른 3자리인지 확인하고 아닐시 IllegalArgumentException 발생시킴

## 계산
스크라이크, 볼, 낫싱 계산하여 출력

## 게임 종료
3개의 숫자를 모두 맞추면 3스트라이크와 게임 종료 문구 출력

## 재시작
게임 종료 후 재시작 또는 종료를 입력할 수 있게 1 또는 2 입력받기
