# 필요한 기능

우선 순서대로 어떤 기능들이 필요할지부터 추려보았다.

- 숫자야구 게임 시작 문구 출력
- 정답 숫자 생성
- 반복문
  - 반복문
    - 올바른 입력인지 확인(3자리 숫자, 0 제외)
    - 숫자 입력 문구 출력
    - 점수 계산(볼, 스트라이크)
    - 점수 출력
    - 3 스트라이크라면 반복문 종료
  - 게임 종료 문구 및 게임 시작/종료 문구 출력
  - 올바른 입력인지 확인(1 또는 2)
  - 2를 입력하면 반복문 종료
- 종료 문구 출력 후 종료.
