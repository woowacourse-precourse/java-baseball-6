### 1. 게임 시작

- [ ]  게임 시작 문구 출력 후 게임 시작 기능
    - `숫자 야구 게임을 시작합니다.`

### 2. 게임 실행

- [ ]  컴퓨터 숫자 생성 기능
    - [ ]  1부터 9까지 랜덤으로 서로 다른 3개 수 생성
        - `camp.nextstep.edu.missionutils`에서 제공하는 `Randoms` API를 사용해야 함
        - 선택한 숫자는 게임 동안 변경되지 않아야 함
- [ ]  유저 숫자 입력 기능
    - [ ]  입력 문구 출력 후 유저로부터 컴퓨터가 생각하고 있는 서로 다른 3개의 숫자를 입력받는 기능

      `숫자를 입력해주세요 :`

    - [ ]  사용자 입력이 유효한지 검증하는 기능
        - 사용자가 입력한 숫자가 세 자리인지 확인
        - 사용자가 입력한 각 자리의 숫자가 모두 다른지 확인
        - 사용자가 입력한 각 자리의 숫자가 1부터 9 사이인지 확인
- [ ]  스트라이크, 볼 판정 기능
    - 숫자가 같고, 자리가 같은 경우 : 스트라이크 1만큼 증가
    - 숫자가 같고, 자리는 다른 경우 : 볼 1만큼 증가
- [ ]  결과 출력 기능
    - 스트라이크와 볼이 0일 경우 : `낫싱` 출력
    - 스트라이크 혹은 볼이 0이 아닌 경우 : `n볼 m스트라이크` 출력 (출력 순서는 볼부터 진행)
    - 스트라이크가 3개일 경우 : `3스트라이크\n3개의 숫자를 모두 맞히셨습니다! 게임 종료` 출력

### 3. 게임 종료 및 재시작

- [ ]  게임이 끝나면 사용자에게 재시작 여부를 출력하는 기능
    - `게임을 새로 시작하려면 1, 종료하려면 2를 입력하세요.`
- [ ]  사용자가 1을 선택하면 게임을 재시작 (컴퓨터가 새로운 3개 숫자 다시 선택), 2를 선택하면 프로그램을 완전 종료하는 기능

### 4. 예외 처리

- [ ]  예외가 발생하면 `IllegalArgumentException` 을 발생시키고 프로그램을 종료하는 기능
