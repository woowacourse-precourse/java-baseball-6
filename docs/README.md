# 기능 목록

## 컴퓨터

* 게임당 랜덤한 서로 다른 3자리 숫자 생성
* 컴퓨터의 숫자와 플레이어가 입력한 숫자를 비교
  * 다 맞추면 게임종료
* 비교한 결과에 따라 출력

## 플레이어

* 컴퓨터의 숫자를 맞추기 위해 서로 다른 3자리수 입력
    * 예외발생 - IllegalArgumentException 후 종료
        * 숫자를 입력하지않았을경우
        * 세자리가 아닌경우
        * 서로 다른 숫자가 아닌경우
* 게임 종료 후 재사작 1 또는 종료 2 를 입력
    * 예외발생
        * 1 또는 2가 아닌 값을 입력 할 경우
