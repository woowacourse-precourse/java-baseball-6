# 1주차 미션: 숫자 야구 게임
## 기능 구현 목록

### 1. 숫자 랜덤 생성 <br>


     1~9 까지의 3가지의 중복되지 않는 숫자


### 2. 입력 예외사항<br>


    예외상황 발생 시 `IllegalArgumentException`을 발생시키고 종료
    * 음수
    * 중복된 숫자
    * 1~9 사이가 아닌 수
    * 종료 입력시 종료 조건에 해당하지 않는 수 


### 3. 입력값과 생성값 비교<br>
    같은 수가 같은 자리에 있으면 스트라이크
    같은 수가 다른 자리에 있으면 볼
    같은 수가 전혀 없으면 낫싱
### 4. 종료 조건
    숫자를 맞춘 후 
    1: 재시작<br>
    2: 종료
